Colander Companion

Collect web content as evidence for your Colander investigations

Abstract

Colander Companion web-extension allows Colander users to gather web content as evidence.

Evidences are uploaded into user Colander account, ready to be converted to Artifacts.

Colander Companion and Colander are parts of PTS Project.
